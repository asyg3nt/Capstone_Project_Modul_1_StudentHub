## Deskripsi Project
Project Ini Bernama **StudentHUB**, aplikasi Python yang saya buat untuk tugas Capstone Project Module 1 Data Science di Purwadhika, aplikasi ini dibuat untuk membantu administrasi data siswa, nilai siswa, menghitung rata rata, hitung raport, dan menentukan ranking siswa per kelas

Program ini menggunakan library tambahan:
* pyinputplus: Untuk validasi input agar program tidak error saat user salah ketik
* tabulate: Untk merapikan tampilan output data menjadi bentuk tabel

## Fitur Aplikasi
Aplikasi dengan sistem CRUD(Create, Read, Update, Delete), dengan pilihan fiturnya:
1. **Registrasi Siswa**: Menginput data siswa baru, serta ID siswa dibuat otomatis supaya tidak terjadi duplikat
2. **Lihat Profil**: Bisa liat data semua siswa atau filter per kelas
3. **Rapor & Ranking**: Otomatis menghitung rata-rata nilai siswa dan mengurutkan rnking dari yang paling tinggi, serta dapat melihat detail nilai persiswanya
4. **Tambah Tugas Baru**: Fitur untk nambahin kolom nilai tugas baru ke semua siswa sekaligus
5. **Update Nilai Borongan**: Input nilai ujian akhir sekaligus, jadi nggak perlu input satu-satu
6. **Hapus Data**: Unutuk ngehapus data siswa dari database

## Cara Menjalankan Aplikasi
1. Sebelum di-run, pastikan untuk install library tambahan yang digunakan:
    
    pip install pyinputplus tabulate

2. lalu jalankan filenya: studenthub.py
